# Skyscanner Registration Date Picker (Frontend)

An intuitive, highly accessible, and responsive date picker component modeled after Skyscanner’s iconic user experience, optimized for user registration and booking flows.

## 🚀 Features

- **Dual-Month View:** Side-by-side or stacked calendar view adaptable to screen breakpoints (optimized for 480px portrait and up).
- **Flexible Selection:** Seamlessly handles range selections (Departure/Return) or single birthday selection for registration.
- **Backpack-Inspired UI:** Employs design tokens, color palettes, and smooth transitions matching Skyscanner's official look and feel.
- **Keyboard Accessible:** Full ARIA compliance with keyboard shortcuts for accessibility.

## 🛠️ Tech Stack

- **Framework:** React 18+ / Vue 3 / Vanilla JS (Choose yours)
- **Styling:** Tailwind CSS / SCSS
- **Testing:** Jest & React Testing Library
